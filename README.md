# Grama-Yatri 🚍

## Community-Powered Rural Bus Tracking System

### 📖 Project Description
Grama-Yatri is an Android-based rural mobility application that provides real-time bus tracking and ETA (Expected Time of Arrival) prediction for village and rural transportation systems.

The application helps passengers track buses using GPS-based live location tracking and community-powered updates. It reduces uncertainty in rural transport where buses often do not follow fixed schedules.

Users can view live bus locations, estimated arrival times, nearby stops, and receive delay notifications directly through the mobile application.
